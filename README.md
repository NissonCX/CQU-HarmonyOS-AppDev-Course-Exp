# CQU-ArkTS-Course-Exp

重庆大学ArkTS课程实验项目

## 项目介绍

本项目是基于HarmonyOS的ArkTS语言开发的小游戏合集，包含了多个经典小游戏：

1. 扫雷游戏 (Minesweeper)
2. 24点游戏 (24 Points)
3. 2048游戏
